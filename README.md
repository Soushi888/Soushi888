## 🧭 What I'm about

Most software concentrates power. I'm interested in the opposite: **agent-centric systems** where
data, governance, and economic value stay with the people who create them. That means Holochain
instead of blockchains, [ValueFlows / REA](https://www.valueflo.ws/) instead of ledgers, and
open-source instead of platforms. I write about it, teach it, and ship it.

- 🔗 **P2P & distributed systems**: Holochain hApps, agent-centric data, local-first
- 📊 **Alternative economics**: hREA, ValueFlows, Open Value Networks, resource governance
- 🤖 **AI-assisted development**: agent skills & tooling that make niche domains accessible to LLMs
- 🌍 **Knowledge as commons**: digital gardens, first-principles pedagogy, writing in public
